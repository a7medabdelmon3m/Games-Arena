# Games Arena - Gaming Platform Landing Page

> A modern, responsive landing page specifically designed for a gaming community platform. This project simulates a real-world client brief to build a pixel-perfect interface using **Bootstrap**.

##  Technologies Used
* **HTML5** (Semantic Structure)
* **CSS3** (Custom Styling & Effects)
* **Bootstrap 5** (Grid System & Components)
* **Responsive Design** (Mobile-First Approach)

##  Key Features
* **Pixel-Perfect Design:** Fully replicated the client's mockup with 100% accuracy.
* **Fully Responsive:** Optimized for all screen sizes (Desktop, Tablet, and Mobile).
* **Modern UI Components:** Utilized Bootstrap cards, navbar, and grid layouts efficiently.
* **Cross-Browser Compatibility:** Ensures consistent performance across different browsers.

##  Advanced Styling (Bonus)
Implemented a special **"Glassmorphism"** effect in the *Our Team* section using CSS `backdrop-filter`.
* **Hover Effect:** When hovering over a team member's card, a blur effect with a semi-transparent overlay is applied, adding a premium, modern feel to the user interface.
